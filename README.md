# bubbleh.com

Static personal homepage for `BubbleWong.github.io`, configured for the custom domain `bubbleh.com`.

## Deploy to GitHub Pages

1. Create or open the `BubbleWong/BubbleWong.github.io` repository.
2. Commit these files to the repository root.
3. In GitHub repository settings, set Pages source to the default branch root.
4. Keep the `CNAME` file in the root so GitHub Pages serves `bubbleh.com`.
5. Point DNS for `bubbleh.com` at GitHub Pages using GitHub's current custom-domain instructions.

## Edit links

Update the social links in `index.html` under the `Find me online` section when you want to add LinkedIn, X, YouTube, Discord, or another profile.
